Frequently Asked Questions

How are the courses structured?

Each course follows a similar format but varies in length:

  • LLM Zoomcamp: 8-10 weeks, focusing on building LLM applications
  • AI Dev Tools: 4-6 weeks, covering various AI-powered development tools
  • Open-Source LLMs: 6-8 weeks, deep diving into model training and deployment

All courses include pre-recorded videos, live office hours, hands-on projects, and community support.

Which course should I take first?

Currently, only the LLM Zoomcamp is running (2nd cohort). The other courses are in development:

  • LLM Zoomcamp: Currently running - join the 2nd cohort now!
  • AI Dev Tools Zoomcamp: Coming soon - sign up to get notified
  • Open-Source LLMs Zoomcamp: In development - sign up to get notified

If you're interested in AI and LLMs, we recommend joining the LLM Zoomcamp now while waiting for the other courses to launch.

What's included in each course?

For the currently running LLM Zoomcamp:

  • Pre-recorded video lessons
  • Weekly live office hours
  • Hands-on projects and assignments
  • Access to course materials and code
  • Active Slack community support
  • Certificate upon completion

The upcoming courses (AI Dev Tools and Open-Source LLMs) will follow a similar format.

Note: All courses are free and open source!

How do I get certified?

To earn a certificate for any course, you'll need to:

  1. Join during an active cohort
  2. Complete the course project(s)
  3. Review 3 peers' projects
  4. Submit all required assignments

Each course has its own certification requirements, detailed on the individual course pages.

What technical setup do I need?

Basic requirements for all courses:

  • Computer with internet connection
  • Python installed (for LLM and Open-Source LLMs courses)
  • VS Code or similar IDE
  • Git for version control

Additional requirements:

  • LLM Zoomcamp: OpenAI API key (optional)
  • AI Dev Tools: GitHub account
  • Open-Source LLMs: GPU access (cloud options available)

Are the courses beginner-friendly?

Each course is designed for different experience levels:

  • LLM Zoomcamp: Beginner-friendly, requires basic Python
  • AI Dev Tools: Suitable for any developer
  • Open-Source LLMs: Intermediate level, requires ML basics

All courses focus on practical implementation over theory, with comprehensive explanations and step-by-step guidance.

How can I engage with the community?

Our vibrant community offers multiple ways to engage:

  • Join our active Slack workspace
  • Participate in weekly office hours
  • Collaborate on projects with peers
  • Share your learning journey on social media
  • Contribute to course materials on GitHub

Use hashtags #llmzoomcamp, #aidevtools, or #osllms to connect with fellow learners!

How often are courses updated?

We regularly update our courses to keep pace with rapid AI developments:

  • Content updates between cohorts
  • New tools and frameworks as they emerge
  • Additional examples and use cases
  • Community contributions and feedback

Once enrolled, you'll have access to all future updates!
